# Parking-Lot-Classifier


This tool allows for a quick classification of busy-free parking spaces in a parking lot using aerial visual data.

## Tools used

- Python
- scikit-learn
- scikit-image
- numpy
- Kaggle

## Data used

- Kaggle dataset: https://www.kaggle.com/datasets/berkaykarabulut/parking-area-dataset
